# Uprawnienia Ról w Systemie Zadaniowo

Dokument opisuje szczegółowo możliwości każdej roli – w trybie **standardowym** (bez Dyrektorki) oraz **z włączoną funkcją Dyrektorki**.

---

## Tryb Standardowy (bez Dyrektorki)

### 1. Administrator (Rola Globalna)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Sprawozdania, Zespoły, Użytkownicy, Archiwum, Ustawienia |
| **Użytkownicy** | Tworzy, edytuje, usuwa konta. Resetuje hasła |
| **Zespoły** | Tworzy zespoły, wybiera kolory, przypisuje koordynatorki, zarządza członkami |
| **Ustawienia** | Pełny dostęp do konfiguracji globalnej (limity, funkcje, uprawnienia) |
| **Zadania** | Widzi zadania wszystkich zespołów, może je korygować/usuwać, może zlecać nowe dowolnym osobom |
| **Ogłoszenia** | Może wstawiać ogłoszenia do wszystkich osób w systemie |
| **Czat** | Może napisać do każdego użytkownika w dowolnym momencie |
| **Sprawozdania** | Widzi raporty wszystkich zespołów |
| **Archiwum** | Może tworzyć nowe foldery i udostępniać je dowolnemu użytkownikowi |
| **Wydarzenia** | Może utworzyć wydarzenie i się na nie zapisywać |
| **Kalendarz** | Widzi spotkania wszystkich zespołów, może dodawać/usuwać spotkania z pełną funkcjonalnością |

### 2. Koordynatorka (Funkcja w Zespole)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Sprawozdania, Zespół, Archiwum, Ustawienia, Dostępne Zespoły, (Aplikacje jeśli włączona rekrutacja) |
| **Zadania** | Tworzy zadania dla zespołu, sprawdza je (akceptuje/odrzuca z uwagami) |
| **Kalendarz Spotkań** | Może tworzyć spotkania dla zespołu |
| **Wydarzenia** | Może przeglądać wydarzenia i brać w nich udział |
| **Ogłoszenia** | Może przeglądać ogłoszenia |
| **Czat** | Może pisać do osób ze swojego zespołu, do innych koordynatorek i do administratorów |
| **Mój Zespół** | Widzi listę osób, może je usuwać i dodawać |
| **Sprawozdania** | Po każdym spotkaniu musi uzupełnić raport |
| **Rekrutacja** | Jeśli włączone, może akceptować nowe osoby (przeglądać Aplikacje) |
| **Archiwum** | Może tworzyć foldery i udostępniać je osobom w swoim zespole, innym koordynatorkom i administratorowi |
| **Ustawienia** | Może włączyć aplikacje do zespołu |

### 3. Uczestniczka (Rola w Zespole)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Archiwum, Dostępne Zespoły |
| **Zadania** | Może wykonywać zadania i poprawiać |
| **Kalendarz Spotkań** | Może przeglądać kalendarz spotkań swojego zespołu i oznaczać obecność |
| **Wydarzenia** | Może przeglądać i zapisywać się na wydarzenia |
| **Ogłoszenia** | Może przeglądać ogłoszenia |
| **Czat** | Może pisać do koordynatorki |
| **Archiwum** | Może tworzyć foldery i udostępniać je koordynatorce |

## Tryb z włączoną funkcją Dyrektorki

### 1. Administrator (Rola Globalna)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Sprawozdania, Zespoły, Użytkownicy, Archiwum, Ustawienia |
| **Użytkownicy** | Tworzy, edytuje, usuwa konta. Resetuje hasła |
| **Zespoły** | Tworzy zespoły, wybiera kolory, przypisuje **dyrektorki**, zarządza członkami |
| **Ustawienia** | Pełny dostęp do konfiguracji globalnej |
| **Zadania** | Widzi zadania zlecone dyrektorką oraz te udostępnione przez koordynatorkę/dyrektorkę. Może sprawdzać, akceptować, odrzucać. Może udostępniać dowolnemu użytkownikowi. Może zlecać wszystkim i widzi zlecone |
| **Ogłoszenia** | Może wysyłać do wszystkich osób w systemie |
| **Czat** | Może napisać do każdego w dowolnym momencie |
| **Sprawozdania** | Widzi raporty wszystkich zespołów |
| **Archiwum** | Może tworzyć foldery i udostępniać dowolnemu użytkownikowi |
| **Wydarzenia** | Może utworzyć wydarzenie i się na nie zapisywać |
| **Kalendarz** | Widzi spotkania wszystkich zespołów, pełna funkcjonalność |

### 2. Dyrektorka (Funkcja w Zespole)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Sprawozdania, Zespół, Archiwum, Ustawienia, Dostępne Zespoły, (Aplikacje jeśli włączona rekrutacja) |
| **Zadania** | Tworzy zadania dla zespołu **oraz dla koordynatorek**. Sprawdza je (akceptuje/odrzuca). Może udostępniać zadania administratorowi lub innym dyrektorkom |
| **Kalendarz Spotkań** | Może tworzyć spotkania dla zespołów, w których jest dyrektorką |
| **Wydarzenia** | Może tworzyć nowe wydarzenia i brać w nich udział |
| **Ogłoszenia** | Może pisać ogłoszenia do zespołów, w których jest dyrektorką |
| **Czat** | Może napisać do każdego użytkownika, administratora, koordynatorek i dyrektorek |
| **Mój Zespół** | Widzi listę osób, może je usuwać/dodawać, **mianować na koordynatorkę** |
| **Sprawozdania** | Ma wgląd do sprawozdań ze spotkań |
| **Rekrutacja** | Jeśli włączone, może akceptować nowe osoby |
| **Archiwum** | Może tworzyć foldery i udostępniać je **wszystkim** |
| **Ustawienia** | Może włączyć aplikacje do zespołu |

### 3. Koordynatorka (Funkcja w Zespole)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Sprawozdania, Zespół, Archiwum, Dostępne Zespoły |
| **Zadania** | Tworzy zadania dla zespołu, sprawdza je (akceptuje/odrzuca z uwagami) |
| **Sprawozdania** | Obowiązkowe uzupełnianie raportu po każdym spotkaniu |
| **Spotkania** | Układa kalendarz spotkań, dodaje linki do rozmów online |
| **Wydarzenia** | Może przeglądać wydarzenia i się na nie zapisywać |
| **Ogłoszenia** | Może wysyłać ogłoszenia do swojego zespołu |
| **Czat** | Może pisać do osób ze swojego zespołu i do **dyrektorki** swojego zespołu |
| **Mój Zespół** | Może usuwać i dodawać nowe osoby |
| **Archiwum** | Może tworzyć foldery i udostępniać je dyrektorkom i administratorowi |

### 4. Uczestniczka (Rola w Zespole)

| Moduł | Uprawnienia |
|---|---|
| **Pulpit** | Zadania, Kalendarz Spotkań, Wydarzenia, Ogłoszenia, Czat, Archiwum, Dostępne Zespoły |
| **Zadania** | Może wykonywać zadania i poprawiać |
| **Kalendarz Spotkań** | Może przeglądać kalendarz swojego zespołu i oznaczać obecność |
| **Wydarzenia** | Może przeglądać i zapisywać się na wydarzenia |
| **Ogłoszenia** | Może przeglądać ogłoszenia |
| **Czat** | Może pisać do koordynatorki |
| **Archiwum** | Może tworzyć foldery i udostępniać je koordynatorce |
